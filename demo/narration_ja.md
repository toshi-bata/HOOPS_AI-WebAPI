# HOOPS AI MCP — デモ ナレーション（日本語）

---

## スライド 1 — Title

「HOOPS AI MCP」をご紹介します。これは、CAD データをインテリジェントに解析するプラットフォームです。CAD ビューアー、B-Rep 解析、加工フィーチャー認識、そして形状類似検索という 4 つの主要機能を、FastAPI による REST API と、Claude Desktop から自然言語で操作できる MCP サーバーの 2 つのインターフェースで提供します。

---

## スライド 2 — Architecture

システムは 4 層構造になっています。土台となるのは Tech Soft 3D 製の HOOPS AI で、CAD ファイルの読み込み・形状解析・機械学習による推論を担います。その上に FastAPI の WebAPI が乗り、HOOPS AI の機能を REST API として公開します。MCP サーバーはその WebAPI をラップし、Claude Desktop が自然言語でツールを呼び出せるようにブリッジする役割を持ちます。そして最上位の Claude Desktop が、AI エージェントとして MCP ツールを自律的に呼び出し、CAD 解析タスクをエンドツーエンドで実行します。

---

## スライド 3 — Features

4 つの機能を順にご説明します。まず「CAD Viewer」は、STEP・SolidWorks・CATIA・NX など 30 以上のフォーマットをブラウザで 3D 表示します。次に「B-Rep Analysis」では、面隣接グラフの生成とフェイス・エッジ属性の抽出が可能です。「MFR」は学習済み ML モデルを使い、穴・スロット・ポケットなど 24 種の加工フィーチャーを自動認識し、結果をビューアー上に色付けで可視化します。最後の「CAD Similarity Search」は、HOOPS Embeddings で形状を特徴ベクトルに変換し、FAISS インデックスで類似形状を高速検索します。

---

## スライド 4 — WebAPI Endpoints

WebAPI は機能ごとにエンドポイントを整理しています。CAD Viewer の起動・終了、B-Rep の面隣接グラフ生成と属性抽出、MFR のファイル検索・サムネイル取得・ラベル一覧・推論実行・ビューアー色付け、そして形状類似検索の合計 11 エンドポイントを提供します。Swagger UI から即座にテスト可能です。

---

## スライド 5 — MCP Tools

MCP Tools は Claude Desktop から呼び出せるツール群です。WebAPI の各機能に対応した 11 個のツールを提供しており、Claude は自然言語の指示だけでビューアーを起動したり、加工フィーチャーを認識したり、類似形状を検索したりできます。エンジニアがコードを書かなくても、対話的に CAD 解析を進められるのが最大の特長です。

---

## スライド 6 — Summary

まとめです。HOOPS AI MCP は、REST API と AI エージェント連携という 2 つのアプローチで CAD インテリジェンスを提供します。B-Rep 解析と加工フィーチャー認識により CAD データを深く理解し、HOOPS Embeddings と FAISS を組み合わせた類似検索ではスコア 0.99 以上の高精度を実証しました。CAD データ活用の新しい可能性を、ぜひご体験ください。
