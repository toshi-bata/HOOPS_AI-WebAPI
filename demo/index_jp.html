HOOPS AI

Tech Soft 3D が提供する 3D CAD 専用の機械学習フレームワーク。30 以上の 3D CAD フォーマットの読み込みから、データセット構築、モデル学習・推論まで、3D CAD と AI に必要なすべてをひとつのツールキットで提供します。

📂 3D CAD データ読み込み

HOOPS Exchange ベースで STEP・SolidWorks・CATIA・NX など 30 以上のフォーマットをネイティブサポート。追加パーサー不要。

🗄️ データセット構築

3D CAD ファイルから形状・属性データを抽出し、ML 学習用データセットをスクリプトで自動生成。

🧠 モデル学習・推論

加工フィーチャー認識・形状類似検索などの学習済みモデルを提供。Python API でカスタムモデルの学習・推論も可能。

🖼️ 可視化

HOOPS Visualize for Web 統合により、推論結果を 3D ビューアー上にリアルタイムで色付け表示。

Features

🖥️ 3D CAD Viewer

STEP・SolidWorks・CATIA・NX など 30 以上の 3D CAD フォーマットをブラウザでインタラクティブに 3D 表示。ファイルアップロードまたは共有パスから起動。

🔗 B-Rep Analysis

B-Rep モデルの面隣接グラフ生成・フェイス/エッジ属性（種類・面積・長さ・二面角等）の抽出。グラフ可視化 (PNG)。

🔍 MFR — Manufacturing Feature Recognition

学習済み ML モデルによる Inference（推論）で、3D CAD ファイルの加工フィーチャ（穴・スロット・ポケット等 24 種）を認識。推論結果をビューアー上に色付けで可視化。

🎯 3D CAD Similarity Search

HOOPS Embeddings モデルで 3D CAD 形状を特徴ベクトルに変換し、FAISS インデックスを使って類似品を高速検索。上位 K 件をスコア付きで返す。
